# Page Topology

## 1. Navigation (Fixed)
- **Overlay:** Overlays all sections.
- **Content:** Logo, Links, CTA Button.

## 2. Hero Section
- **Layout:** Full-height, video background.
- **Interaction:** Scroll-triggered reveal.

## 3. About Section
- **Layout:** Sticky container.
- **Content:** Large text, "About Us" heading.

## 4. Work Section
- **Layout:** Grid structure.
- **Typography:** Anton font.
- **Content:** Project cards.

## 5. Why Us Section
- **Layout:** Image background.
- **Content:** Key selling points.

## 6. Testimonial Section
- **Layout:** Carousel or list.
- **Content:** User reviews.

## 7. Contact Section
- **Layout:** Form based.
- **Typography:** Anton font.

## 8. Footer
- **Layout:** Standard footer content.
- **Interaction:** Revealed as page ends.
